Stpm2

stpm2

stpm2 fits flexible parametric survival models. These models use splines to model some transformation of the survial function. The most common is the $\log[-\log[S(t)]]$ link function, which fits proportional hazards models. I have added some examples and aim to add to these. Proportional hazards models comparison with a Cox model simple simulation study to show agreement with Cox model. predicting hazard and survival functions (use of the timevar() option) sensitvity analsis for the number of knots.
